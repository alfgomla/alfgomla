# Alfonso Gómez — Portfolio

Portafolio personal moderno y minimalista construido con React, Vite y Tailwind CSS.

## 🚀 Tecnologías

- React 19
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## 📦 Instalación

```bash
npm install
npm run dev
```

## 🌐 Despliegue en GitHub Pages

```bash
npm run deploy
```

> Asegúrate de tener configurado `gh-pages` y que el repositorio remoto apunte a `https://github.com/alfgomla/alfgomla.git`

## 📁 Estructura

```
src/
  components/
    Navbar.jsx
    Hero.jsx
    About.jsx

    Experience.jsx
    Education.jsx
    Skills.jsx

    Footer.jsx

  main.jsx
  index.css
```
